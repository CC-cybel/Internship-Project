# 指标对比表格生成 (Metric Comparison Table)

## 技能描述
此技能用于快速生成多个模型的关键评估指标对比表格，直观展示各模型在核心维度上的表现差异，并自动高亮最优值。

## 适用场景
- 需要在报告、PPT 或群聊中快速展示多个模型的横向对比结果时。
- 关注重点在于 **Average Score (平均分)**、**Hard Pass Rate (硬性通过率)**、**Sample Pass Rate (完美率)** 和 **Rule Macro Pass Rate (平均规则通过率)** 这四个核心指标。

## 输入要求
- **评估结果文件路径**：提供所有参与对比模型的 `evaluation_report.md` 文件路径。
  - 例如：`/data1/yezj/gitlab/leadbench/output/model_name_version/evaluation_report.md`

## 输出格式
输出为一个 Markdown 表格，包含以下列：
1. **模型 (Model)**：模型名称。
2. **Average Score (平均分)**：综合评分。
3. **Hard Pass Rate (硬性通过率)**：无严重错误（扣分>1）的样本比例。
4. **Sample Pass Rate (完美率)**：得分为 100 分的样本比例。
5. **Rule Macro Pass Rate (平均规则通过率)**：所有规则通过率的平均值。

**样式要求**：
- 表头需包含中文注释，例如 `**Average Score**<br>(平均分)`。
- **每一列的最高数值必须加粗显示**（例如 `**97.26**`）。
- 百分比保留两位小数。

## 示例输出

| 模型 | **Average Score**<br>(平均分) | **Hard Pass Rate**<br>(硬性通过率) | **Sample Pass Rate**<br>(完美率) | **Rule Macro Pass Rate**<br>(平均规则通过率) |
| :--- | :--- | :--- | :--- | :--- |
| **Model A** | 95.99 | 67.55% | 42.48% | 87.05% |
| **Model B** | **97.26** | **72.57%** | **47.79%** | **92.87%** |
| **Model C** | 93.13 | 28.61% | 16.81% | 88.53% |

## 操作步骤
1. **读取数据**：从每个模型的 `evaluation_report.md` 中提取四个核心指标数值。
2. **数据清洗**：确保所有百分比格式统一（保留两位小数）。
3. **寻找极值**：遍历每一列数据，找到最大值。
4. **格式化输出**：构建 Markdown 表格，将最大值用 `**` 包裹进行加粗。
