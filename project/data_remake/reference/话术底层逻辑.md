# 话术底层逻辑

> **一句话核心**：所有话术都是在调节对方的神经激素，激素决定状态，状态决定决策。

---

## 一、生物层 — 你在操控什么

| 激素 | 感受 | 触发条件 | 行为结果 |
|-----|------|---------|---------|
| **皮质醇** | 危机/紧迫 | 感知到威胁或损失 | 理性↓，行动力↑ |
| **多巴胺** | 期待/好奇 | 预期到奖励（未得到时最强） | 主动追求，愿意投入 |
| **催产素** | 信任/被理解 | 感受到真诚连接 | 防御↓，愿意分享 |
| **血清素** | 被尊重/有地位 | 被认可、被抬高 | 配合意愿↑ |
| **内啡肽** | 轻松/愉悦 | 笑声、问题被解决 | 好感↑，记忆锚定 |

**三条使用原则**：
- 皮质醇是双刃剑，医疗用户基线本来就高，**适度激活=紧迫感，过度=逃跑**
- 多巴胺靠不确定性，**确定的奖励不如"可能的奖励"**刺激更大
- 催产素是留资的前置条件，**没有信任基础，其他所有机制都是沙上建塔**

## 二、心理层 — 激素如何变成可利用的杠杆

### 1. 互惠原则
**激活**：催产素

先给予价值 → 对方产生亏欠感 → 留联系方式成为"还债"的自然出口。

关键：**给予越具体越个性化，债务越重**。"我们提供免费咨询"产生的债，远小于"我专门帮你把刚才说的症状整理了一下时间线"。

```
给予价值 → 等对方表达感谢/认同 → 再提请求
× 错误：给完立刻索要联系方式（交易感出现，互惠感消失）
```

---

### 2. 社会认同
**激活**：血清素（不被排斥）+ 皮质醇↓

不确定时，大脑默认"别人做的=安全的"。核心是**近似性原则**：越像自己的人做了这件事，说服力越强。

```
弱："我们服务了5万名患者"（抽象）
强："上周有个妈妈，孩子也是7岁多，
    她当时也观察了两个月才来问，
    后来评估发现骨龄已经提前了——
    她说庆幸当时没再等。"（具体+近似+故事）
```

---

### 3. 权威效应
**激活**：血清素 + 皮质醇↓

大脑遇到权威自动关闭批判系统，节能走捷径。**权威要精准，不能泛化**。

```
弱："我们有专业医生"
强："我们有一位主任，在儿童医院做内分泌20年，
    她说家长描述的情况，往往比实际轻——
    因为家长会本能地往好处想。"
```

---

### 4. 承诺与一致性（登门槛）
**激活**：血清素（自我形象一致性）

每次引导对方说出一个"是"，都在强化他的内部自我形象。当形象够强，留联系方式成为**维持这个形象的自然选项**。

```
"现在方便聊吗？" → "孩子几岁了？" → "什么时候开始的？"
→ "您最希望得到什么样的结论？"
→ 对方说："就想知道孩子正不正常"
→ "那最直接的方式是让医生评估，我帮您约，留个联系方式就好"
```

让对方**自己说出目标**，比你替他说效果强10倍。

---

### 5. 损失厌恶
**激活**：皮质醇↑

损失的痛苦是等值收益的2.25倍。医疗场景中健康损失不可逆，比金钱更有力。**本质不是吓唬，是帮对方看见他已经在承受的隐性代价**。

```
弱（收益框架）："留个联系方式，获得免费评估"
强（损失框架）："您说观察了两个月了——
               这两个月里骨龄可能已经在悄悄提前。
               骨龄一旦闭合，身高是没办法补回来的。"
```

⚠️ **用一次效果最大，用两次对方感觉被操控，用三次对方逃跑。**

---

### 6. 稀缺性
**激活**：多巴胺 + 皮质醇

稀缺资源大脑自动标注高价值。分三种，DISC反应不同：

| 类型 | 示例 | 适用人群 |
|-----|------|---------|
| 数量稀缺 | "本周名额只剩3个" | D型有效，S型反感 |
| 时间稀缺 | "7岁还在干预窗口期，过了8岁方案完全不同" | 所有类型 |
| 专属稀缺 | "这位主任档期很紧，我帮你看看这周还有没有" | I/S型最有效 |

## 三、战术层 — 具体招式

### 招式地图

| 招式 | 目标激素 | 核心机制 | 一句话描述 |
|-----|---------|---------|---------|
| 提问术 | 多巴胺+催产素 | 承诺升级 | 用问题引导对方自己走向留资 |
| 共情术 | 催产素 | 社会连接 | 说出对方没说出口的感受 |
| 互惠术 | 催产素 | 互惠原则 | 先给具体价值，再提请求 |
| 权威术 | 血清素 | 权威效应 | 精准背书，关闭对方批判系统 |
| 案例术 | 多巴胺+催产素 | 社会认同 | 用近似故事激活镜像神经元 |
| 锚定术 | 皮质醇 | 锚定效应 | 先说高成本，再说低门槛 |
| 稀缺术 | 多巴胺+皮质醇 | 稀缺性 | 制造错过感，但不能滥用 |

---

### 各招式要点

**提问术** — 漏斗结构

```
宽："您主要是为孩子咨询吗？"
 ↓
中："孩子多大？什么情况让您担心？"
 ↓
窄："大概从什么时候开始的？"
 ↓
核："您最希望得到什么样的结论？"  ← 让对方说出自己的目标
 ↓
转："那最直接的方式就是..."
```

---

**共情术** — 三个层次，层次越深催产素越多

```
L1 表层（无效）："我理解您的担心"
L2 内容层：    "您观察得这么仔细，两个月的变化都记得这么清楚"
L3 深层（最强）："发现孩子可能有问题，又不知道严不严重，
                 这种不确定比知道答案更难受。"
                 → 对方："对！就是这个感觉"→ 催产素大量分泌
```

共情后必须转向，不能停留：**共情 → 正常化 → 给价值 → 引导行动**

---

**锚定术** — 先高后低

```
"完整的儿科内分泌评估要预约门诊、做检查、复诊，
 可能两三个月才有结论；（高锚）
 我们可以先安排一个电话评估，
 20分钟给您一个初步方向。"（低要求）
 → 锚定"两三个月"，电话变得极其轻松
```

---

**案例术** — 四要素缺一不可

```
相似性：  "有个妈妈，孩子也是7岁多..."（对方能代入）
具体性：  "也是胸部有硬块"（不是"症状类似"这种模糊表达）
过程感：  "她一开始也觉得不严重，后来..."（有转折）
结果感：  "干预之后发育恢复正常了"（给希望但不保证）
```

## 四、三层联动：每轮对话的执行流程

```
Step 1  读取状态
        └─ 用户当前哪种激素失衡？（皮质醇过高/催产素不足？）
        └─ 三账户哪个余额最低？（情绪/风险/价值）
        └─ DISC类型是什么？

Step 2  确定本轮目标
        └─ 我要调节哪种激素？
        └─ 我要充哪个账户？

Step 3  选择招式 + 组织语言（控制在3句以内）
        句1：共情接住（催产素铺垫）
        句2：核心内容（本轮选择的招式）
        句3：轻引导（小承诺或行动方向）

Step 4  检查红线
        └─ 是否触碰用户雷区？
        └─ 是否有催促/推销语气？
        └─ 皮质醇是否已经过载？
```

---

## 五、DISC对应策略速查

| 类型 | 核心需求 | 账户权重 | 最有效招式 | 绝对禁忌 |
|-----|---------|---------|---------|---------|
| **D型** | 结果、效率 | 价值>风险>情绪 | 锚定+权威 | 啰嗦、不给结论 |
| **I型** | 被认可、希望 | 情绪>价值>风险 | 共情+案例 | 冷漠、数据堆砌 |
| **S型** | 安全、不被催 | 风险>情绪>价值 | 互惠+社会认同 | 催促、早期索要联系方式 |
| **C型** | 专业、逻辑 | 价值>风险>情绪 | 权威+具体数据 | 空洞表达、无法验证的承诺 |
