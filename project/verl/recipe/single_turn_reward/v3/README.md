# single_turn_reward/v3

`v3/` contains the third staged single-turn reward recipe. In the current project history it is associated with later pre-contact or mid-stage experiments and older stage-5-mid reward files.

## Purpose

This folder helps preserve the transition from simpler contact-stage reward logic to more nuanced middle-stage training. It is useful when comparing how prompts, reward-model wrappers, and run scripts evolved.

## Typical Contents

- reward functions for intermediate-stage behavior;
- rule-only variants for debugging;
- reward-model wrappers;
- launch scripts for 4-GPU Qwen3 GRPO jobs;
- tests or utilities for validating reward outputs.

## Notes

Many active ideas from this folder were later moved or rewritten in newer locations. Check the launch script paths and input dataset paths before running anything.
